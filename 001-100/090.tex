\section{90 --- Subsets II}
Given a collection of integers that might contain duplicates, $A$, return all possible subsets (the power set).
\par
Note: The solution set must not contain duplicate subsets.
\paragraph{Example:}
\begin{flushleft}
\textbf{Input}: [1,2,2]
\\
\textbf{Output}:
\\
$(2),(1),(1,2,2),(2,2), (1,2), ()$
\end{flushleft}
