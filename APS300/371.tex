\section{371 --- Sum of Two Integers}
Calculate the sum of two integers $ a $ and $ b $, but you are not allowed to use the operator $ + $ and $ - $.

\paragraph{Example 1:}

\begin{flushleft}
\textbf{Input}: $ a = 1 $, $ b = 2 $
\\
\textbf{Output}: 3
\end{flushleft}

\paragraph{Example 2:}

\begin{flushleft}
\textbf{Input}: $ a = -2 $, $ b = 3 $
\\
\textbf{Output}: 1
\end{flushleft}

\subsection{Bit Operation}
Get carry with \fcj{a & b}
Get XOR result with \fcj{a ^ b}.
Set \fcj{b} as \fcj{carry} left shift 1.

Notice: in some compilers, we cannot left shift $-1$. A way to bypass this issue is to get carry with \fcj{a & b & 0x7fffffff} to clear the highest bit.

\setcounter{lstlisting}{0}
\begin{lstlisting}[style=customc, caption={Bit Operation}]
int getSum( int a, int b )
{
    while( b )
    {
        //some compiler do not support left shift -1
        //we need to clear highest bit with 0x7fffffff
        int carry = ( ( a & b ) & 0x7fffffff );
        a = a ^ b;
        b = ( carry << 1 );
    }
    return a;
}
\end{lstlisting}
